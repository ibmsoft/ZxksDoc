        # 用户注册

        1、选择 我的 进入 如下页面

        2、点击登录，进入如下页面

        3、选择右上角的 注册，进入注册页面

        4、输入账号和密码

        TIP

        账号，可以用个人的手机号，这样方便记忆

        5、选择人员类型 6、选择单位类型 7、选择区域，单击河南省，到自己所在区域

        TIP

        完成相关上面的信息，整个用户就注册完成了
